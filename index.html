About

I'm a full stack developer with a passion for creating clean, modern, mobile ready web applications that put the user experience first.

Prior to web development I worked in ecommerce, first as a marketplace sales specialist and later as an account manager for large platform clients, serving Amazon, Ebay, Walmart, and more.

I have always been passionate about software development and the user experience which I believe are essential to providing easy and simple access to education and improving one's quality of life.

  • Dance Up

    Dance Up is a software as a service (SaaS) web application built with Ruby on Rails. Other technologies used are jQuery, Bootstrap, and SCSS. Users can create a free account with limited user information access or a paid account with full access. Additionally, users can upload avatars and post status updates in their profiles. Payments are captured through Stripe. Mailers are handled by Sendgrid.
